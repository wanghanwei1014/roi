# 用户使用指南

## 1. 功能介绍

广告 ROI 数据分析系统提供基于时间维度的投资回报率趋势可视化功能，帮助用户分析移动应用广告投放效果。

### 核心功能

- 查看 5 款应用（App-1 ~ App-5）的 ROI 趋势
- 支持 8 个时间维度（当日、1日、3日、7日、14日、30日、60日、90日）
- 按国家（美国、英国）筛选数据
- 移动平均线平滑趋势展示
- 基于历史数据的 ROI 预测
- 100% 回本线参考标识

## 2. 数据导入操作指南

### 首次使用

1. 确保后端服务已启动（默认端口 3001）
2. 通过以下任一方式导入数据：

**方式一：API 导入（推荐）**

```bash
curl -X POST http://localhost:3001/api/import
```

**方式二：命令行导入**

```bash
cd backend
npm run import-csv
```

3. 导入成功后会显示导入的记录数（预期 910 条）

### 更新数据

重新执行导入命令即可。系统会先清空旧数据再导入新数据。

## 3. 图表操作指南

### 3.1 趋势线

图表默认显示 8 条 ROI 趋势线，每条线代表不同的 ROI 统计周期：

| 颜色 | 含义 |
|------|------|
| 紫色 | 当日 ROI |
| 绿色 | 1日 ROI |
| 橙黄色 | 3日 ROI |
| 橙色 | 7日 ROI |
| 蓝色 | 14日 ROI |
| 青色 | 30日 ROI |
| 红色 | 60日 ROI |
| 紫色（深） | 90日 ROI |

### 3.2 100% 回本线

图表中有一条红色虚线标注 100%，表示广告投入回本的阈值。ROI 超过此线意味着广告投放已实现盈利。

### 3.3 预测数据

图表末尾以虚线样式展示未来 7 天的预测数据，基于最近 14 天的趋势通过线性回归计算得出。

### 3.4 数据点悬停

将鼠标悬停在图表的任意位置，会显示该日期所有 ROI 维度的具体数值。预测数据点会有"(预测)"标注。

### 3.5 图例交互

点击图表下方的图例项可以显示/隐藏对应的趋势线。被隐藏的线条在图例中会显示为灰色删除线样式。

### 3.6 关于 0% 数据

系统智能区分两种 0% 数据：
- **真实 0%**：该日期的实际 ROI 确实为 0%，图表正常显示
- **日期不足 0%**：数据统计周期不足导致的 0%，图表中该数据点被隐藏（不显示、不连线）

## 4. 筛选功能使用方法

页面顶部提供 4 个筛选控件：

### 4.1 用户安装渠道

固定为 "Apple"，当前版本不可更改。

### 4.2 出价类型

可选择 "CPI"（每次安装成本）或 "全部"。当前数据集中出价类型统一为 CPI。

### 4.3 国家地区

- **全部**：同时显示美国和英国的数据
- **美国**：仅显示美国投放数据
- **英国**：仅显示英国投放数据

### 4.4 APP

选择要分析的应用：App-1、App-2、App-3、App-4 或 App-5。切换后页面标题和图表数据会相应更新。

## 5. 控制器使用说明

### 5.1 数据显示模式

- **显示移动平均值**（默认）：使用 7 日移动平均值平滑数据，减少日常波动，更容易观察趋势
- **显示原始数据**：显示每日的实际 ROI 数值

### 5.2 Y 轴刻度

- **线性刻度**（默认）：Y 轴按等间距显示百分比
- **对数刻度**：Y 轴按对数比例显示，适合数据量级差异较大时使用（如 90 日 ROI 远高于当日 ROI 的情况）

## 6. 暗色模式

点击页面右上角的太阳/月亮图标可切换亮色/暗色模式。图表和所有 UI 组件会自动适应当前主题。
